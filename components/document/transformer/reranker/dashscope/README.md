# DashScope Reranker

## 功能介绍

DashScope Reranker 是 Eino 框架的一个文档重排序组件，基于阿里云 DashScope 平台提供的文本重排序 API 实现。该组件可以对一组文档按照与查询的相关性进行重排序，提高检索结果的准确性和相关性。

## 注意事项

1. 使用前需要在阿里云 DashScope 平台获取 API 密钥。
2. 请参考 [DashScope 文本重排序 API 文档](https://help.aliyun.com/zh/model-studio/text-rerank-api?spm=a2c4g.11186623.help-menu-2400256.d_2_11_0.77857bbdIe6pFI) 了解支持的模型和参数详情。
3. 不同模型可能支持的参数和功能有所差异，请根据实际使用的模型调整参数配置。
4. API 调用可能产生费用，请参考 DashScope 平台的计费标准。
